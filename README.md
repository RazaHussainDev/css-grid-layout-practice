# CSS Grid Layout Practice

A simple learning assignment to practice basic HTML and CSS Grid Layout concepts.

## Learning Objective

- Learn how to structure a webpage using HTML.
- Practice building multi-column and multi-row layouts with CSS Grid.
- Understand CSS grid properties like `grid-template-columns`, `grid-template-rows`, `gap`, and `grid-column` / `grid-row`.
- Experiment with box coloring, spacing, and alignment in a grid system.

## Technologies Used

- HTML5
- CSS3 (with focus on CSS Grid)

## Features

- Simple grid-based layout with colored boxes
- Responsive design using viewport units
- Basic HTML and CSS only—no JavaScript

## Folder Structure

```
css-grid-layout-practice/
├── index.html
├── style.css
└── README.md
```

## How to Open / Run the Project

1. **Download or Clone the repository** to your computer.
2. Open `index.html` in any web browser (Chrome, Firefox, Edge, etc.).
3. You should see a grid of colored boxes created using CSS Grid.

## Notes

- This project is for educational purposes and practice only.
- All code is very basic and beginner-friendly.
- Feel free to experiment by changing grid properties in `style.css`.

## Future Learning Goals

- Try adding more rows/columns to the grid.
- Practice using CSS Grid for real-world layouts (like navbars, footers, etc.).
- Combine CSS Grid with Flexbox for more complex designs.
- Learn about media queries for responsive layouts.
