# CSS Grid Layout Practice

A simple learning assignment to practice basic HTML and CSS Grid Layout concepts.

## Learning Objective

- Learn how to structure a webpage using HTML.
- Practice building layouts with CSS Grid.
- Understand main grid properties like `grid-template-columns`, `grid-template-rows`, `gap`, `grid-column`, and `grid-row`.
- Experiment with box coloring and spacing in a grid.

## Technologies Used

- HTML5
- CSS3 (CSS Grid layout)

## Features

- Simple grid-based layout with different colored boxes
- Responsive design using viewport units
- No JavaScript, only HTML and CSS

## Folder Structure

```
css-grid-layout-practice/
├── index.html
├── style.css
└── README.md
```

## How to Open / Run the Project

1. **Download** or **clone** this repository to your computer.
2. Find the file named `index.html`.
3. Double-click `index.html` to open it in your web browser (such as Chrome, Firefox, or Edge).
4. You will see a grid of colored boxes.

## Notes

- This project is for learning and practice only.
- Code is kept very simple for beginners.
- Try changing values in `style.css` to see different grid layouts.
- Feel free to ask your teacher if you are confused about any part!

## Future Learning Goals

- Add more rows or columns and try different grid settings.
- Use CSS Grid for more realistic layouts (like headers and footers).
- Combine CSS Grid with Flexbox.
- Learn about responsive layouts using media queries.
